Tiles Using SVG

Gives you huge amount of control. Also gives you anything SVG can do, including gradients and other funky affects (not that it's clear we need those affects). But have to use math to position everything exactly, and may be limited in using icons -- may eat up too much processing power

Tiles Using Divs

Can use external icons, images, etc. Can also use anything that uses HTML, such as the Bootstrap CSS Framework, which also gets you responsive design without a lot of work. Might also give us a cleaner way of doing Queues and maps that doesn't require a lot of math (although to be fair, the math for simple SVG maps, for example, is pretty straightforward).
